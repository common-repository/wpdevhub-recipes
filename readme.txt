== Description ==

Host Recipes on your WordPress Website

== Installation ==

1. Unzip the contents of `wpdevhub-recipes.zip`
2. Upload the full `wpdevhub-recipes` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Access the plugin through the links on the left of WordPress.  Widgets and short codes are also available
